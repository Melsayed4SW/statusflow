Privacy Policy for StatusFlow

Effective Date: May 2026

Overview

StatusFlow is a browser extension that helps users update calendar work statuses in Google Calendar and Microsoft Outlook Calendar.

Data We Access

StatusFlow may access:

  • Google Calendar events
  • Microsoft Outlook calendar events
  • Authentication tokens required for calendar access
  • Locally stored user preferences

How Data Is Used

Data is used solely to authenticate the user and create or update calendar events requested by the user.

Data Storage

StatusFlow stores authentication tokens locally in the user's browser using Chrome extension storage.

StatusFlow does not transmit user calendar data to external servers owned by the developer.

Data Sharing

StatusFlow does not sell, share, or transfer user data to third parties.

User Control

Users can revoke access at any time by:

  • Logging out of the extension
  • Revoking app access from Google Account settings
  • Revoking app access from Microsoft Account settings
  • Uninstalling the extension
